# YepSVGSampleApp

A minimal iOS SwiftUI sample app that renders the bundled `badge.svg` using `YepSVG`.

## Included SVG

This sample bundles:

`/Users/climbatize/Documents/Dev/yepngo-assets/Badges/rewards/2025/badge.svg`

Note: the directory is `rewards` (plural).

## Run

1. Open:
   `Examples/YepSVGSampleApp/YepSVGSampleApp.xcodeproj`
2. Select scheme: `YepSVGSampleApp`
3. Run on an iOS Simulator (iOS 15+).

## Regenerate project (optional)

If you edit `project.yml`, regenerate the project:

```bash
cd Examples/YepSVGSampleApp
xcodegen generate
```
